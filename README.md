# Simulyn

An AI-powered API resilience simulation platform that lets developers 
test how their APIs behave under real-world failure conditions — before 
real users experience them.

## What it does

Simulyn lets you fire hundreds of concurrent requests at any API endpoint 
while injecting real-world failure conditions like latency spikes, random 
errors, traffic bursts, and database bottlenecks. After each simulation run, 
Gemini AI generates a plain-English resilience report explaining what broke, 
why it broke, and how to fix it.
